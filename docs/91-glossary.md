# 91. Glossary

- Runtime: 모델 실행 엔진
- Harness: 도구와 작업 루프를 제공하는 에이전트 시스템
- Stack: 모델·런타임·하네스·도구 조합
- GGUF: llama.cpp 계열 모델 형식
- Quantization: 저비트 가중치 압축
- TTFT: 첫 토큰 시간
- TPS: 초당 생성 토큰
- KV Cache: 컨텍스트 Attention 상태
- Tool Calling: 외부 도구 호출 생성
- Worktree: 격리된 Git 작업 디렉터리
- RAG: 검색 자료를 컨텍스트로 제공
- Hard Fail: 즉시 탈락하는 보안·무결성 위반
