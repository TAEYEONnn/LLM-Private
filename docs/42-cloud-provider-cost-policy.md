## 후보
Ollama Cloud Free 비교용, OpenCode Zen PAYG, OpenRouter PAYG, Native Codex 기준선, Claude 선택적.

## 비용
월 예산 하드 캡, 자동 충전 Off, 자동 폴백 Off, 공급자 고정, 작업당 실제 비용 기록. 초기 PAYG는 월 10달러부터 시작한다.

2개월 연속 PAYG가 구독료보다 높을 때 구독형을 검토한다. 연간 결제는 금지한다.
