# 91. Roadmap v4 Decisions

작성 기준일: 2026-06-28

- V4-001 OpenKnowledge PUBLIC Pilot을 Local LLM보다 먼저 수행한다.
- V4-002 ChatGPT 연결은 읽기·검색부터 시작한다.
- V4-003 작업 완료 Hook은 전체 대화가 아니라 결과·검증·결정만 기록한다.
- V4-004 Agent가 생성한 문서는 기본적으로 Draft다.
- V4-005 Approved 문서는 직접 덮어쓰지 않고 변경 제안을 만든다.
- V4-006 자동 Git Push와 자동 유료 승격을 금지한다.
- V4-007 최소 2주 또는 작업 30건을 측정한 뒤 Local 필요성을 판단한다.
- V4-008 Local LLM은 초기 필수 구성이 아니라 조건부 Executor다.
- V4-009 Local 구축이 보류돼도 ChatGPT·Codex·OpenKnowledge Workflow는 유지한다.
- V4-010 기존 `LLM-Private` 문서는 OpenKnowledge Pilot 승인 후 선별 이관한다.
